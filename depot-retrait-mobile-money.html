Dépôt et retrait sur 1xBet avec Mobile Money

L’un des grands avantages de la plateforme en Afrique francophone (Cameroun, Côte d'Ivoire, Sénégal, etc.) est la possibilité de gérer vos fonds directement avec votre téléphone portable, de manière rapide et sécurisée.

Effectuer un Dépôt Mobile Money

Pour pouvoir placer un pronostic ou activer un avantage, vous devez d'abord approvisionner votre compte.

  1. Connectez-vous à votre compte. Si vous n'en avez pas, apprenez comment créer un compte ici.
  2. Allez dans la section "Faire un dépôt" (le bouton vert en haut).
  3. O Orange Money
  4. M MTN Mobile Money
  5. Moov, Wave, etc.
  6. Entrez le montant souhaité et votre numéro de téléphone.
  7. Confirmez la transaction via le code secret sur votre téléphone. L'argent apparaît quasi instantanément.

Rappel Bonus de Dépôt

N'oubliez pas, pour doubler votre premier dépôt, utilisez le code promo MB25T à l'inscription. Découvrez les détails sur le bonus 1xBet !

Comment Faire un Retrait ?

Retirer ses gains est tout aussi simple, mais une règle stricte s'applique : vous devez utiliser la même méthode pour le retrait que celle utilisée pour le dépôt.

  1. Dans votre profil personnel, cliquez sur "Retirer des fonds".
  2. Sélectionnez votre méthode Mobile Money.
  3. Indiquez le montant de vos gains à retirer.
  4. Renseignez le numéro de compte / téléphone associé.
  5. La validation prend entre 15 minutes et 24 heures selon la congestion du réseau.

Délais et Problèmes Fréquents

Dans la majorité des cas, les dépôts sont instantanés et les retraits prennent moins d'une heure. S'il y a un retard de retrait, cela peut venir d'une vérification de compte non complétée. En cas de doute prolongé, nous avons compilé un guide des problèmes dans notre page Foire Aux Questions (FAQ) où nous abordons les retards.
